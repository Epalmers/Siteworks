# Data Center Site Selection Model

This project evaluates potential locations for data centers using a weighted multi-criteria decision analysis (MCDA) model.

## Features
- Uses ranked environmental and economic data
- Allows user-defined weighting of risk categories
- Outputs ranked city recommendations

## How to Run
1. Install pandas
2. Run:
   python site_selection_model.py

## Inputs
- CSV file with ranked scores for each category

## Output
- Ranked list of cities based on total weighted score
